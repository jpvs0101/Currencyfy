# Currencyfy
[Android] [Java]   Converts number to readable currency format. Easy, error free &amp; highly compatible!

currencyfy (500000.78) -> $ 500,000.78

currencyfy (500000.78, false) -> $ 500,001

currencyfy (500000.78, false, false) -> 500,001
